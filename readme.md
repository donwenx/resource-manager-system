## 资源文件管理系统
---
### 功能：
1. 资源分类管理
2. 资源管理（上传、下载、审核、发布）
3. 资源权限管理
4. 资源使用情况统计
5. 用户注册、导入: 普通用户、超级管理员

---
### 食用方法

#### 1. 启动方法

```bash
git clone https://gitee.com/zijiu7/resource-manager-system.git

# 下载后端依赖
npm install
# 下载前端依赖
cd client
npm install
cd ..
# 启动命令
npm run dev

```

#### 2. 配置数据库

修改`10`行到`13`行为自己的`MySQL`数据库账号、密码、数据库名

登录请使用`UID`+`密码`登录
登录需要注册，注册后点跳转会自动把`uid`填写到登录`id`处
